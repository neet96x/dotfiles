# Skill: trade-journal

1. บันทึกต่อดีล (ลงไฟล์ csv/md): วันเวลา, สินทรัพย์, ทิศทาง (long/short), ขนาด, ราคาเข้า/ออก, **เหตุผลที่เข้า**, **เหตุผลที่ออก**, ผล (กำไร/ขาดทุน %), บทเรียน
2. **จด thesis ก่อนรู้ผล**: เขียนเหตุผล+แผน (TP/SL) ตอนเข้า ไม่ใช่มาแต่งทีหลัง — กัน hindsight bias
3. ทบทวนเป็นรอบ (สัปดาห์/เดือน): หา pattern ที่ทำกำไร vs ที่พลาดซ้ำๆ, วินัยหลุดตรงไหน
4. แยก **กระบวนการดี vs ผลดี**: ดีลที่ขาดทุนแต่ทำตามแผน = โอเค; ดีลที่กำไรแต่มั่ว = อันตราย
5. **ซื่อสัตย์กับการขาดทุน** — บันทึกตามจริง ไม่ลบดีลแย่ทิ้ง
6. ถ้ามีไฟล์ journal อยู่แล้ว อ่านก่อนแล้ว append รูปแบบเดิม

ไม่ให้คำแนะนำการลงทุน — ช่วยจด/วิเคราะห์สิ่งที่ผู้ใช้ทำเท่านั้น
