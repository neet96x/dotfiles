# Skill: summarize

1. อ่าน**ทั้งหมด**ก่อน (read_file; ไฟล์ยาวอ่านเป็นช่วง) — อย่าสรุปจากแค่ต้นเรื่อง
2. จับ: ประเด็นหลัก, โครงสร้าง, ข้อสรุป/ข้อเสนอ, และ action item ถ้ามี
3. รูปแบบผลลัพธ์:
   - **TL;DR**: 1 ประโยค
   - **ประเด็นหลัก**: bullet 3-7 ข้อ
   - **สิ่งที่ต้องทำ/ตามต่อ**: ถ้ามี
4. ซื่อตรงต่อต้นฉบับ — ห้ามเติมข้อมูลที่ไม่มี, ห้ามตีความเกิน, ระบุถ้าต้นฉบับขัดแย้งกันเอง
5. ปรับความยาวตามที่ขอ ("สั้นๆ" = 2-3 บรรทัด)

ถ้าเป็นโค้ด: สรุปว่าทำอะไร + โครงสร้างหลัก (ดู skill explain-code)
